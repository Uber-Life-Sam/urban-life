# Urban Life – GTA Style Update

This update adds new gameplay features:

### ✔ Mouse Camera Movement
The player can rotate the camera like GTA using mouse movement.

### ✔ GTA-Style Player Movement
Smooth third-person movement system similar to GTA games.

### ✔ Player House & Land
A house area for the player that detects when the player enters the property.

### ✔ Traffic Light System
A 4-way traffic light that changes automatically.

### ✔ Vehicle AI
Simple AI cars that stop at signals and move when the light turns green.
